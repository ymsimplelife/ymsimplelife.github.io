簡單生活 環保生活

環保新選擇 - 滑鼠出現連點或者拖曳不完全問題，先別急著丟掉滑鼠，微動修理經驗分享

Jun 02, 2023
1 minute read

心得

緣起

  • 滑鼠出現連點以及拖曳不完全現象。
  • 但是滑鼠一隻幾千，捨不得直接丟棄。

找解決方式

  • 上網查詢後，發現是「微動」壞掉。
  • 看了很多經驗分享，更換微動也不算簡單的事情，需要焊接槍等工具才能施作完畢。
  • 於是決定找看看有沒有商家提供相關服務。

找到信賴店家

  • 很幸運，發現了士林的「極致工藝電腦醫學中心」，網路評價優良，處理過的案件也多。
  • 透過 Line 與老闆預約時間後，直接帶滑鼠過去修。
  • 老闆會先很仔細說明流程才進行修理。
  • 修理時，會說明進行到哪一個階段以及滑鼠狀態。
  • 大概經過 30 分鐘，就完成更換微動了。

現況

  • 已再持續使用兩個月。

總結

  • 超級推薦「極致工藝電腦醫學中心」。
  • 優點一：環保
    • 不必丟棄整個滑鼠，畢竟滑鼠的其他零件都還可以使用
  • 優點二：省錢
    • 關於報價可以參考「極致工藝電腦醫學中心」的部落格
